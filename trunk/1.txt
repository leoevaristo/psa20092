Como utilizar o TortoiseSVN -


1. Como criar um repositório para armazenar os arquivos vindos dp Google Code?? Clica com o botão direito encima da pasta e seleciona a opção Criar Repositório.

2. Como baixar os arquivos do google code para o seu repositório??
 Abrir a pasta do repositório e utilizar o comando checkout. Utilizando a URL passada no site do google code: 
 
https://psa20092.googlecode.com/svn/trunk
